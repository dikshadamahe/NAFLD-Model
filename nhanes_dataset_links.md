## Variables Used Per Dataset

### DEMO_J (Demographics)
| NHANES Variable | Renamed To | Description |
|-----------------|------------|-------------|
| RIDAGEYR | Age | Age in years at screening |
| RIAGENDR | Gender | Gender (1=Male, 2=Female) |
| RIDRETH3 | Ethnicity | Race/Hispanic origin with NH Asian |

### BMX_J (Body Measurements)
| NHANES Variable | Renamed To | Description |
|-----------------|------------|-------------|
| BMXBMI | BMI | Body Mass Index (kg/m²) |
| BMXWAIST | Waist_Circumference | Waist circumference (cm) |

### TRIGLY_J (Triglycerides)
| NHANES Variable | Renamed To | Description |
|-----------------|------------|-------------|
| LBXTR | Triglycerides | Triglycerides (mg/dL) |
| LBDLDL | LDL | LDL Cholesterol, calculated (mg/dL) |

### HDL_J (HDL Cholesterol)
| NHANES Variable | Renamed To | Description |
|-----------------|------------|-------------|
| LBDHDD | HDL | Direct HDL-Cholesterol (mg/dL) |

### GLU_J (Fasting Glucose)
| NHANES Variable | Renamed To | Description |
|-----------------|------------|-------------|
| LBXGLU | Glucose | Fasting Glucose (mg/dL) |

### BIOPRO_J (Standard Biochemistry)
| NHANES Variable | Renamed To | Description |
|-----------------|------------|-------------|
| LBXSATSI | ALT | Alanine Aminotransferase (U/L) |
| LBXSASSI | AST | Aspartate Aminotransferase (U/L) |
| LBXSCH | Total_Cholesterol | Total Cholesterol (mg/dL) |

*All datasets are in SAS Transport (.XPT) format. Load in Python with `pandas.read_sas()`.*
