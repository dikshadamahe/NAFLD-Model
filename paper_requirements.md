This repository contains a machine learning project for predicting Non-Alcoholic Fatty Liver Disease (NAFLD).

The final research paper must replicate the structure and style of sample_paper.pdf.

Required components:

Tables:
- Table 1: Literature comparison
- Table 2: Hyperparameters and their domains
- Table 3: Dataset characteristics
- Table 4: Model performance comparison
- Table 5: Cross-validation results

Algorithms:
- Algorithm 1: Dataset preprocessing and splitting
- Algorithm 2: NAFLD model training pipeline
- Algorithm 3: NAFLD prediction pipeline

Figures:
- Dataset distribution graph
- ROC curve comparison
- Confusion matrix
- Feature importance
- SHAP summary plot
- Methodology flowchart

Evaluation metrics:
Accuracy
Precision
Recall
F1-score
ROC-AUC

Statistical tests:
McNemar test

Cross validation:
5-fold stratified cross validation

The output must be a full research paper with no page limit in LaTeX format.
